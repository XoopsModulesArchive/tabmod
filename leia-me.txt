Install Instructions:

A) Install the Theme
1) Upload the theme into your theme directory (ucberkeley_liquid).  
2) Go to Admin > Sysadmin > Preferences > Selectable themes > select 
ucberkeley_liquid and any other theme you want (you must have the themes block 
enabled) 
3) Alternate: Admin > Sysadmin > Preferences > Default theme
4) Use the default template or a cloned default template


B) Install the Module
1) Upload the tab mod module in the module directory
2) Install the module
